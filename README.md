# Music Store Sales Analysis — SQL

## Project Overview
Analysed a digital music store database using SQL to identify 
top performing genres, artists, customers and monthly sales trends.

## Tools Used
- DBeaver — SQL editor
- SQLite — database

## Key Findings
- Rock is the top genre with $826.65 in total revenue
- Iron Maiden is the best selling artist with $138.60 in revenue  
- USA has the most customers (13) but Czech Republic has 
  the highest spending customer (Helena Holý — $49.62)
- January 2010 was the best performing month with $52.62 revenue

## Queries Written
1. Customers by country
2. Top 5 customers by spending
3. Revenue by genre
4. Monthly sales trend
5. Top 10 artists by revenue
